1- que es bootstrap

    es un marco front-end gratuito para un desarrollo web más rápido y fácil
    también le brinda la capacidad de crear fácilmente diseños receptivos

    Diseño web responsive = diseno adaptable a cualquier tipo de pantalla

2- Instalaar bootstrap

    <!-- Latest compiled and minified CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">

    <!-- Latest compiled JavaScript -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script>

    <!-- etiqueta que configura que la pagina sea responsiva -->
    <meta name="viewport" content="width=device-width, initial-scale=1">



3- Conocer etiquetas de bootstrap

    existen dos tipos de contenedores:
    -CONTENEDOR NORMAL hay un espacio en los laterales de la pagina
    -CONTENEDOR FLUIDO  agarra toda la pagina

    Bootstrap va partir la pagina en 12 contenedores (GRID)
